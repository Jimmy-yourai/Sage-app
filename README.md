## 🚀 DEPLOY IN 4 STEPS

### STEP 1 — Upload to GitHub
1. Go to https://github.com → Sign up / Login
2. Click "+" → "New repository"
3. Name: `sage-app` → Public → Create
4. Click "uploading an existing file"
5. Drag & drop ALL files from this ZIP:
   - index.html, signup.html, login.html, app.html
   - style.css, particles.js, jimmy.jpg, README.md
6. Click "Commit changes"

### STEP 2 — Deploy on Vercel
1. Go to https://vercel.com → Sign up with GitHub
2. Click "Add New Project"
3. Select `sage-app` repository
4. Click "Deploy"
5. ✅ LIVE in 30 seconds!

Your URL: `https://sage-app.vercel.app`
